##  708. 偶数

### 题目

编写一个程序，输出 $1$ 到 $100$ 之间（包括 $1$ 和 $100$）的全部偶数。

### 输入格式

无输入。

### 输出格式

输出全部偶数，每个偶数占一行。

### 数据范围



### 输入样例

```
No input
```

### 输出样例

```
2
4
6
...
100
```

### AC代码

```c++
#include <cstdio>

int main()
{
    for(int i=1;i<=100;i++){
        if(i%2==0) printf("%d\n",i);
    }
    return 0;
}
```
