##  714. 连续奇数的和 1

### 题目

给定两个整数 $X$ 和 $Y$，输出在他们之间（不包括 $X$ 和 $Y$）的所有奇数的和。

### 输入格式

第一行输入 $X$，第二行输入 $Y$。

### 输出格式

输出一个整数，表示所有满足条件的奇数的和。

### 数据范围

$-100≤X,Y≤100$

### 输入样例

```
6
-5
```

### 输出样例

```
5
```

### AC代码

```c++
#include <cstdio>

int main()
{
    int a,b,n=0;
    scanf("%d%d",&a,&b);
    if(a > b){
        int c=a;
        a=b;
        b=c;
    }
    for(int i=a+1;i<b;i++){
        if(i%2){
            n+=i;
        }
    }
    printf("%d",n);
    return 0;
}
```
